# HTML & CSS: Bootstrap Grid Layout

Esercizio di replica di un layout responsive tramite il **Grid System di Bootstrap 5**. Il focus del progetto è l'implementazione di una struttura flessibile che si adatta a diversi viewport utilizzando le classi utility del framework.



## 🎯 Obiettivo
Testare le conoscenze su:

* **Struttura semantica HTML** integrata con componenti Bootstrap.
* **Sincronizzazione dei Breakpoint**: Gestione del passaggio da layout a colonna singola (mobile) a multi-colonna (desktop).
* **Griglie Annidate**: Utilizzo di `.row` dentro altre `.col` per layout complessi.

## 🛠️ Tecnologie Utilizzate
* **HTML5**: Strutturazione dei contenuti.
* **CSS3**: Per eventuali personalizzazioni di stile e colori.
* **Bootstrap 5.3**: Core framework per la gestione della responsività.
